# rindexer
index contract events for EchoFi

## local development
run the docker container for rindexer locally
```bash
cd rindexer/echofi-rindexer-uni-v3
docker compose up -d
```

## spin up the service
```bash
rindexer start indexer
```

## deploy the service
just push to master => [Railway](https://railway.app/template/Rqrlcf)
